    Cara install tema semi online
    pertama buka link wordpress.org/themes pada browser, download tema, terdapat file zip dan disimpan, kemudian pilih tema tambah baru, unggah tema, choose file, pilih file Zip dan install. Jika tidak cukup ruang taruh skrip
    - php_value upload_max_filesize 128M >php_value post_max_size 128M. php_value max_execution_time 300 php_value max_input_time 300
    Masukan ke htacsess. Reload dan dapat berguna untuk file tidak terlalu besar
